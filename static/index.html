Соглашение на обработку персональных данных

Настоящее соглашение разработано в соответствии с законодательством о защите персональных данных и регулирует порядок обработки персональных данных, предоставляемых пользователями в рамках использования приложения для профориентационного тестирования (далее – «Приложение»).

1. Определение персональных данных и цели их обработки

1.1. В рамках использования Приложения обрабатываются следующие персональные данные пользователей:

  • Номер телефона (при указании пользователем);
  • Результаты профориентационного тестирования.
2. Принципы обработки персональных данных

2.1. Обработка персональных данных осуществляется на основе принципов законности, конфиденциальности, минимизации данных и их безопасности.

2.2. Приложение не собирает и не обрабатывает данные, не связанные с достижением указанных целей (например, ФИО, адреса, дату рождения и т.п.).

3. Порядок и условия обработки персональных данных

3.1. Персональные данные пользователей обрабатываются с использованием автоматизированных и неавтоматизированных средств.

3.2. Доступ к персональным данным имеют только уполномоченные сотрудники или партнеры, задействованные в обеспечении функционирования Приложения и связанных услуг.

4. Хранение и защита персональных данных

4.1. Для защиты персональных данных пользователей принимаются необходимые организационные и технические меры, направленные на предотвращение несанкционированного доступа, уничтожения, изменения и распространения данных.

5. Права пользователей

5.1. Пользователи имеют право на:

  • Получение информации о целях и способах обработки их персональных данных;
  • Запрос на исправление, блокировку или удаление предоставленных данных;
  • Отзыв согласия на обработку персональных данных.

5.2. Для реализации своих прав пользователи могут обратиться по контактам, указанным в Приложении.

6. Заключительные положения

6.1. Продолжая использовать Приложение, пользователь выражает согласие с настоящим соглашением и дает разрешение на обработку своих персональных данных в рамках установленных целей.

6.2. Приложение оставляет за собой право вносить изменения в настоящее соглашение. Обновленная версия соглашения будет опубликована в Приложении и вступит в силу с момента публикации.
